# Imposter Client

React application built with Vite.

## Tech Stack
- React 18
- React Router 6
- Axios
- Socket.io Client
- CSS Modules / Vanilla CSS

## Key Components
- `Sidebar`: Main navigation
- `EmailList` / `EmailView`: Email interface
- `SmsList`: Chat-style SMS interface
- `CardView`: 3D CSS-only credit card component

## Running Tests

```bash
npm test
```
